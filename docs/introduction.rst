Introduction
============

.. include:: includes/all.rst

The ``ypid.plymouth`` role allows you configure a graphical boot and shutdown animation using
`Plymouth <https://wiki.debian.org/plymouth>`_.


Installation
~~~~~~~~~~~~

This role requires at least Ansible ``v2.1.4``. To install it, run:

.. code-block:: console

   ansible-galaxy install ypid.plymouth
